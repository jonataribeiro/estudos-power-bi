# 📊 Estudos de Power BI

Repositório dedicado ao registro da minha evolução, estudos e projetos práticos utilizando o **Power BI**. O objetivo aqui é centralizar a criação de dashboards dinâmicos, extração de dados e modelagem de negócios.

---

## 🛠️ Tecnologias e Ferramentas
* **Power BI Desktop**
* **Linguagem DAX** (Expressões de Análise de Dados)
* **Power Query** (Linguagem M para ETL)
* **Modelagem Dimensional** (Star Schema e Snowflake)

---

## 📁 Estrutura do Repositório

* `├── Introduca_power_BI/` → Conceitos iniciais, conexões de dados e primeiros relatórios.
* `├── Introducao_DAX/` → Fórmulas, funções calculadas, medidas e inteligência de tempo.
* `├── DevImport/` → Projetos práticos e desafios de importação/transformação de dados (ETL).

---

## 🚀 O que estou praticando aqui?
* **ETL Eficiente:** Limpeza, tratamento e transformação de dados brutos utilizando o Power Query.
* **Cálculos Avançados:** Criação de medidas e colunas calculadas otimizadas com DAX para responder a perguntas de negócio.
* **Design e UI/UX:** Construção de dashboards visualmente atraentes, intuitivos e focados na tomada de decisão rápida.
